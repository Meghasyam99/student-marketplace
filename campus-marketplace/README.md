# Campus Marketplace (Frontend UI)

Modern, student-focused marketplace UI built with React + Bootstrap.

## Tech
- React (Vite)
- Bootstrap 5 + Bootstrap Icons
- React Router
- React Toastify

## Pages
- Landing
- Login
- Registration
- Home
- Product Listing
- Product Details
- Add Product
- User Profile
- My Listings
- Wishlist
- Admin Dashboard

## Run locally
```bash
npm install
npm run dev
```

## Notes
- This is a UI-only build with mock data in `src/data/mockData.js`.
- Wishlist + theme mode are persisted in `localStorage`.
- Ready to connect to Django REST APIs later (listings, auth, wishlist, uploads).
